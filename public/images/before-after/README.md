# Before/After Images

This directory contains before and after comparison images for the slider component.

## Required Images

Place the following images in this directory:
- `before.jpg` - The "before" image for the comparison slider
- `after.jpg` - The "after" image for the comparison slider

## Image Requirements

- Format: JPG or PNG
- Recommended size: 1600x900 pixels (16:9 aspect ratio)
- Both images should have the same dimensions for best results
- File size: Keep under 500KB for optimal loading

## Current Usage

The before/after slider is displayed in the About section of the homepage.
